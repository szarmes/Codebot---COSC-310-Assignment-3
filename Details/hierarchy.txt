inheritance,~ In a hierarchy Object-oriented programming allows classes to inherit variables and methods from the extended parent class.,~

structure,~ In the Java, each class is allowed to have one direct superclass, and each superclass has the potential for an unlimited number of subclasses.,~

syntax,~ At the beginning of your class declaration, use the extends keyword, followed by the name of the class to inherit from. Eg. class mountainBike extends bicycle.,~
